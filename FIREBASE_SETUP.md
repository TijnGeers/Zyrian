# Firebase Setup - 2 minuten

## Stap 1: Ga naar Firebase Console
1. Ga naar: https://console.firebase.google.com
2. Log in met je Google account
3. Klik op "Add project"

## Stap 2: Maak project
1. Project naam: `zyrian-vertaler`
2. Klik "Continue"
3. Klik "Create project" (wacht even)

## Stap 3: Activeer Database
1. Klik op "Build" -> "Realtime Database" in het menu
2. Klik "Create Database"
3. Kies "Europe (europe-west1)"
4. Kies "Start in test mode" (voor nu)
5. Klik "Enable"
